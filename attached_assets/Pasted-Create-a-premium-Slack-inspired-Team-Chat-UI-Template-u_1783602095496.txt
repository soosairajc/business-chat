Create a **premium Slack-inspired Team Chat UI Template** using **Tailwind CSS** only. This is a **frontend-only template** for ThemeForest. **Do not build any backend, authentication, APIs, database, Firebase, or server-side functionality.** Focus entirely on a polished, reusable, responsive UI with realistic sample data.

## Design Requirements

* Modern SaaS design with a unique visual identity (do not copy Slack)
* Tailwind CSS (latest version)
* Fully responsive (Desktop, Tablet, Mobile)
* Light & Dark mode
* RTL Ready
* Glassmorphism and soft shadows where appropriate
* Clean typography and spacing
* Component-based structure
* Reusable UI components
* Heroicons or Lucide Icons
* Smooth transitions and animations
* Accessible UI
* ThemeForest-quality code and folder structure
